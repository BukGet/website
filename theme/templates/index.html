Minecraft

BukGet's API is designed to aide system administrators and developers of the popular game Minecraft.

Python

The API is written entirely in Python, a fast & flexible language with a lot of capability. Furthermore, all of the code that is used to run BukGet is open source (just click the code link to goto our github page)!

Bukkit

While the API is designed to interact with multiple plugin datastores, our primary use is as an interactive API into BukkitDev, the plugin repository for the Bukkit Minecraft server.
